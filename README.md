这是一个实时声音特征识别app，实时比对外部声音和样本声音的相似度，将足够相似的声音记录下来

1. 使用现代的android设计规范进行设计
2. 安装在红米、vivo手机上
3. 程序支持长时间后台运行，做好权限设置和性能优化，做好内存和文件限制
4. 程序结构需要清晰，低耦合高内聚
5. 程序分为三个tab，实时、历史、配置
   1. 实时是实时显示处理结果，展示统计次数，实时分析的相似度和阈值，以及最新的日志。停止处理后，可以手动查看数据
   2. 历史显示预警历史
   3. 配置支持录音和上传音频文件，配置相似度和间隔时间，配置ha上传地址，还有数据量相关的设置
4. 支持配置ha地址，实时上传识别结果
